# Introduction

## Version

-   Node: v18.12.1
-   Yarn: v1.22.19

## Functions

-   FrameWork: React + ReactRouter + TypeScript
-   Build: Vite
-   Css: Tailwindcss + Scss
-   UI: Antd
-   Code specification: prettier + eslint
-   Request: Axios
-   State Management: Zustand

# Procedure

```sh
# Install dependencies
yarn

# Start Dev
yarn dev

# Preview from mobile phone => will get a local IP address
yarn run dev --host

# build
yarn run build
```
